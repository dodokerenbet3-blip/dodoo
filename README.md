# ⚔️ RPG Adventure By Rivaldo

Selamat datang di **RPG Adventure By Rivaldo**, sebuah game RPG 3D berbasis Python yang menggabungkan elemen eksplorasi, sistem level, dan cuaca dinamis. Game ini dibuat menggunakan **Ursina Engine** dengan fokus pada performa yang ringan namun tetap memiliki visual yang menarik.

## ✨ Fitur Utama
- **🌍 Open World Exploration**: Jelajahi dunia dengan kamera *Third Person* (Orang Ketiga).
- **⚔️ Combat System**: Sistem pertarungan *Real-time* melawan Slime.
- **⛈️ Dynamic Weather**: Perubahan cuaca otomatis (Hujan & Malam) untuk suasana lebih dramatis.
- **📈 Leveling & XP**: Kalahkan musuh untuk menaikkan level pahlawanmu.
- **💾 Auto-Save System**: Progres permainan tersimpan otomatis dalam file teks.
- **⛩️ Dungeon Portal**: Berpindah dari dunia terbuka ke dalam gua rahasia.

## 🚀 Cara Menjalankan
1. Pastikan kamu sudah menginstal Python di komputermu.
2. Instal library **Ursina**:
   ```bash
   pip install ursina
   ```
3. Jalankan file utama:
   ```bash
   python main.py
   ```

## 🎮 Kontrol Game
- **W, A, S, D**: Bergerak & Rotasi.
- **Klik Kiri**: Menyerang dengan Pedang.
- **F**: Berinteraksi dengan NPC.
- **Q**: Menggunakan Skill Ultimate (Terbuka di Level 2).
- **ESC**: Menampilkan kursor / Keluar.

## 🛠️ Teknologi yang Digunakan
- **Bahasa Pemrograman**: Python 3.x
- **Game Engine**: Ursina Engine
